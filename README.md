# Карточка игрока
## name
Может меняться только игроком
## health point
Может меняться ГМ или взаимодействии
## power
Может меняться только ГМ
## initiative
Может меняться только ГМ
## speed
Может меняться только ГМ
## lvl
Может меняться только ГМ
